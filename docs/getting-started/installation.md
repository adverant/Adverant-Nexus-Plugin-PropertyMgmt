# Installation

## Prerequisites

- Node.js 20+
- npm or yarn
- PostgreSQL 14+
- Redis

## Installation Steps

### Via Nexus Marketplace (Recommended)

```bash
nexus plugin install nexus-<plugin-name>
```

### Manual Installation

```bash
npm install @adverant/nexus-<plugin-name>
```
